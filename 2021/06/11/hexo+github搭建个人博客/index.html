hexo+github搭建个人博客

发表于 2021-06-11 更新于 2021-06-12 阅读次数： Valine：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

注册github账号
 地址：https://github.com/

下载安装node.js
 地址：https://nodejs.org/zh-cn/  默认安装即可
安装完成后，终端输入node -v  出现版本号即node安装成功
终端输入npm -v  出现版本号即npm安装成功

下载安装Git
地址：https://git-scm.com/downloads  默认安装即可
安装完成，运行Git Bash
设置user.name和user.email配置信息 

git config --global user.name "github用户名"
git config --global user.email "github注册邮箱"

生成ssh密钥文件：

ssh-keygen -t rsa -C "github注册邮箱"

进入.ssh目录：windows在C:\Users\用户名\.ssh
查看公钥

cat .\id_rsa.pub

复制公钥信息，登录github，点击头像=>Settings=>SSH and GPG keys=>NEW SSH key
title：随便填写
key：粘贴复制的公钥信息
点击Add SSH key
添加完成后，命令行执行，测试ssh连接是否正常

ssh git@github.com

在github中新建repository，点击头像左侧的 + =>New repository
Repository name：
github用户名.github.io
点击Create repository
浏览器访问：github用户名.github.io
终端cd进入要存blog文件的磁盘根目录，例如：D:
使用npm 命令安装hexo

npm install -g hexo-cli

安装成功后，创建本地blog主目录，“Blog”目录名可自定义

hexo init Blog

win10系统可能默认不允许执行脚本，需要修改策略组

set-executionpolicy remotesigned

再次执行hexo init Blog，成功生成Blog文件夹
cd 进入Blog文件夹
在Blog目录内执行

hexo g

编译生成静态文件；
执行

hexo s

启动预览服务
使用本地浏览器访问localhost:4000 预览博客页面
显示正常时，CTRL+C 停止预览
修改Blog目录下_config.yml文件
文件末尾修改为：

deploy:
 type: git
 repo: git@github.com:用户名/用户名.github.io.git
 branch: main

修改完成后，命令行安装hexo-deployer-git

npm install hexo-deployer-git --save

安装完成后执行：

hexo d

部署并自动上传更新到github新建的repository中
上传成功：访问用户名.github.io
修改github=>Your repositories=>点击【用户名.github.io】=>Settings=>Pages=>
Source修改Branch：main=>Save
重新执行hexo d等待1分钟左右，再访问https://用户名.github.io

添加阿里云域名：
添加一条：iPv4 [ping 用户名.github.io]得到的ip地址 主目录为@
再加一条：CNAME 用户名.github.io 主目录为www

github添加域名：
github=>Your repositories=>点击【用户名.github.io】=>Settings=>Pages=>
Custom domain 添加阿里云购买的域名地址 Save
